RGB Color Space

RGB stands for Red, Green, and Blue, which are the primary colors of light used to create colors on electronic displays like monitors and screens. In the RGB color space, colors are represented as combinations of varying intensities of red, green, and blue. Each color channel typically ranges from 0 (no intensity) to 255 (maximum intensity). For example, the color green is created by setting the green channel to a high intensity (e.g., 255) and the red and blue channels to low intensities (e.g., 0).

HSV Color Space

HSV stands for Hue, Saturation, and Value. It is an alternative representation of colors that separates the color information (hue), the vividness or intensity of the color (saturation), and the brightness (value). Hue (H) represents the actual color itself, and it is measured in degrees around a color wheel (0° for red, 120° for green, 240° for blue, and everything in between). Saturation (S) measures the purity or vividness of a color. A saturation value of 0 represents a shade of gray, while higher values represent more vivid colors. Value (V) represents the brightness of the color. A value of 0 is black, and higher values are progressively brighter.
